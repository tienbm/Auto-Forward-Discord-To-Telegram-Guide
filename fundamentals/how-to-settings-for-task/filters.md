#### **Overview**

* **Purpose**: Allows users to specify which types of messages the bot should ignore during forwarding.
* **Instructions**: Toggle the filter for each message type:
  * **✓ (Green)**: Skip this type of message during forwarding.
  * **✕ (Gray)**: Allow this type of message to be forwarded.

***

#### **Filter Types**

* **FILTERS PHOTOS**: Skip messages containing photos if enabled.
* **FILTERS ALBUM**: Skip messages containing albums or media collections if enabled.
* **FILTERS POLL**: Skip messages containing polls if enabled.
* **FILTERS DOCUMENT**: Skip messages containing files or documents if enabled.
* **FILTERS LINK**: Skip messages with hyperlinks if enabled.
* **FILTERS VIDEO**: Skip messages containing videos if enabled.
* **FILTERS AUDIO**: Skip audio messages if enabled.
* **FILTERS STICKER**: Skip sticker messages if enabled.
* **FILTERS TEXT**: Skip plain text messages if enabled.
* **FILTERS FORWARDS**: Skip forwarded messages if enabled.
* **FILTERS REPLY**: Skip reply-type messages if enabled.
* **FILTERS ANIMATION (GIF)**: Skip GIF animations if enabled.
* **FILTERS VIDEO NOTE**: Skip video notes if enabled.
* **FILTERS BUTTON**: Skip messages containing interactive buttons if enabled.
* **FILTERS PHOTO WITH TEXT**: Skip photos with accompanying text if enabled.
* **FILTERS WITHOUT USERNAME**: Skip messages sent by users without a username if enabled.
* **FILTERS OUTGOING**: Skip outgoing messages (sent by the bot itself) if enabled.
