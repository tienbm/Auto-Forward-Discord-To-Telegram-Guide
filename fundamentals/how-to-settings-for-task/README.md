**This is interface of detail Task**

<figure><img src="../../.gitbook/assets/image (198).png" alt=""><figcaption></figcaption></figure>

**Forwarding Status**: Indicates whether the forwarding is active. In this case, it’s **RUNNING**.

**Show Forward Header**: Specifies if the forward header is displayed (here it is set to **NO**).

**Source**: Displays the **Discord Channel ID** where messages are being forwarded from (e.g., `1330086886491947102`).

**Target**: Displays the **Telegram Chat ID** where messages are being forwarded to (e.g., `-1001716791967`).

**Target Topic**: Indicates the specific topic in Telegram for targeted forwarding (e.g., `8888`).

#### **Edit Task Section**

* **Edit Source**: Modify the Discord channel ID for the source.
* **Edit Target**: Change the Telegram chat ID for the destination.
* **Edit Target Topic**: Update the specific topic in the Telegram chat/group.

#### **Task Control Section**

* **Stop Forward**: Stops the message forwarding process immediately.
* **Delete Task**: Permanently deletes the forwarding task.

**Features**

**Message Forward Control**

* **Show Forward Header**:\
  Toggles the display of the forward header for messages being forwarded.
* **Edit**:\
  Allows you to modify the forward header details.
* **Delete**:\
  Deletes forwarded messages selectively.
* **Reply**:\
  Enables the bot to reply directly to messages in the destination channel.

***

#### **Modify Content**

* **Replace**:\
  Configure specific words or phrases in the forwarded messages to be replaced.\
  **Status**: Shows how many replacements are currently active (e.g., Activated: 0).
* **Add Footer**:\
  Add a footer to the forwarded messages.\
  **Status**: Displays whether a footer is added (e.g., "FOOTER EMPTY").
* **Add Header**:\
  Add a header to the forwarded messages.\
  **Status**: Displays whether a header is added (e.g., "HEADER EMPTY").

***

#### **Filters**

* **Blacklist**:\
  Prevent specific messages containing certain words or phrases from being forwarded.\
  **Status**: Shows the number of active blacklist items (e.g., Activated: 2).
* **Whitelist**:\
  Only allow messages containing specific words or phrases to be forwarded.\
  **Status**: Shows how many whitelist items are active (e.g., Activated: 0).
* **Filters**:\
  Apply specific content-based filters to the forwarded messages.\
  **Status**: Shows the number of active filters (e.g., Activated: 0).
* **Cleaner**:\
  Cleans forwarded messages by removing unwanted elements like formatting or specific parts of the content.\
  **Status**: Shows how many cleaning rules are applied (e.g., Activated: 0).
